I am Sam

I do not like green eggs and ham.

To be honest, I only use this picture because it's been associated with my Github for so long I just won't change it. It was taken at a picnic in Roma Street Parklands that Bec organised. But moving on...

I'm a software engineering and physics graduate starting my PhD this year. My main interests lie in cosmology, machine learning, and generally just doing things right. Perhaps that's the software engineer in me speaking, but a half implemented pipeline or an analysis that requires more than someone pressing the enter key to start seems to be to be fundamentally lacking.

My PhD project is to perform a Bayesian analysis of supernova cosmology. To get myself across everything properly, I hope to update this site with some good tutorials for starting Bayesian analysis with Python, because there is nothing like writing a tutorial to highlight all the little things you actually don't understand.
